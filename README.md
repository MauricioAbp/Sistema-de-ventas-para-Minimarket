# 🛒 Sistema de Ventas para Minimarket

## 📖 Descripción

Este proyecto consiste en un **Sistema de Ventas para Minimarket** desarrollado con una arquitectura **Full Stack**, diseñado con el objetivo de optimizar y facilitar la gestión de las operaciones más importantes dentro de un negocio. La aplicación permite administrar de manera eficiente productos, clientes y ventas, ofreciendo una interfaz moderna e intuitiva que mejora la experiencia del usuario.

La solución está dividida en dos partes principales. El **backend**, desarrollado con **Spring Boot** y **Java**, se encarga de procesar la lógica de negocio y exponer una API REST para la comunicación con el cliente. Por otro lado, el **frontend**, construido con **Angular**, proporciona una experiencia dinámica e interactiva mediante el uso de **TypeScript**, **HTML** y **CSS**.

Este proyecto fue realizado con la finalidad de aplicar conocimientos de desarrollo Full Stack, arquitectura por capas y consumo de servicios REST, utilizando tecnologías ampliamente empleadas en el ámbito profesional.

---

## ✨ Funcionalidades principales

🔹 Gestión y administración de productos.

🔹 Registro y administración de clientes.

🔹 Registro y control de ventas.

🔹 Comunicación entre frontend y backend mediante API REST.

🔹 Interfaz amigable y dinámica desarrollada con Angular.

🔹 Organización del proyecto siguiendo una arquitectura escalable y mantenible.
